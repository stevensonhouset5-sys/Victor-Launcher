Victor Launcher

Build and share room-based mod packs for Among Us.

Create a room, upload DLLs, save the pack, and share the room code. Players only need to enter that code in Victor Launcher in game to queue the right mods.

Create Room

Start a new room for your group and lock it with a password.

Open Room

Log back in later with the room code and password.

How It Works

Everything needed to manage a room pack in one place.

Upload the DLLs you want players to receive. Victor Launcher will queue them in game after a player enters the room code.
Changes are not live until Save Room Pack is pressed. Uploading files alone does not update the room pack.
Removing a DLL here removes it from the room pack and deletes it from the room's file list.
